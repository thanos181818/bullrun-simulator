# **App Name**: BullRun Trading Simulator

## Core Features:

- Real-Time Stock/Crypto Tracking: Fetches and displays real-time stock and crypto data using the Finnhub API.
- ML-Based Simulated Trading: Simulates stock prices using an ML-driven model, offering a risk-free trading environment. Uses stochastic models with Random Walk and Drift algorithms.
- Full Trading Engine: Allows users to buy and sell stocks/crypto within the simulated environment. Manage portfolios and track transaction history.
- Gamification & Leaderboard: Implements a leaderboard and badge system to reward and encourage active participation and learning. Badge awarding is automatically triggered upon completion of tasks (first trade, profit target, complete all modules, streak badge).
- Learning Modules & Quizzes: Provides educational modules and quizzes to teach users about trading strategies and market analysis. Automatically tracks users progress. Questions will assess knowledge on a multitude of topics
- Real-Time Chat: Enables real-time chat functionality for users to communicate and share insights. Built with Socket.IO for live updates.
- AI-Powered Trading Insights: Leverages generative AI to analyze market trends and provide personalized trading suggestions. The suggestions serve as a tool to inform trading decisions within the simulated mode.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Teal (#008080) to highlight interactive elements and key information.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean and neutral design.
- Use sharp, minimalist icons related to finance, trading, and learning modules.
- Maintain a professional stock-market style dashboard layout with sharp edges and well-organized information.
- Subtle animations for price updates and trade executions to provide a dynamic user experience.
